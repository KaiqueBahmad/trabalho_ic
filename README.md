# O Peso da Coroa

Projeto final da disciplina de Introdução à Computação do curso de Matemática Aplicada e Computacional da UFG.

RPG textual em C onde o jogador interpreta um rei governando um reino em crise diplomática e militar, com suporte a narração por voz e acessibilidade para deficientes visuais.
