# Guia do Jogador — O Peso da Coroa

Você é o Rei de Avalon. O reino de Drakmar ameaça suas fronteiras e há suspeitas de traição na corte. O peso da coroa é seu para carregar — e as decisões são todas suas.

## Como Jogar

O jogo é totalmente em texto. Digite um comando e pressione Enter. Não há menu de cliques, tudo é feito pelo teclado.

## Comandos Globais

Estes comandos funcionam em qualquer momento do jogo:

| Comando      | O que faz                                      |
|--------------|------------------------------------------------|
| `ajuda`      | Mostra a lista de comandos                     |
| `status`     | Exibe seus atributos atuais (vida, ouro, etc.) |
| `inventario` | Lista os itens que você carrega                |
| `examinar`   | Descreve todos os itens do inventário          |
| `repetir`    | Repete a última narração por voz               |
| `opcoes`     | Repete as opções do local atual                |
| `salvar`     | Salva o progresso em arquivo                   |
| `carregar`   | Carrega o jogo salvo (perde progresso atual)   |
| `audio`      | Ativa ou desativa a narração por voz           |
| `sair`       | Encerra o jogo                                 |

## Atributos

- **Vida** — Sua saúde. Se chegar a zero, o reino cai.
- **Ouro** — Seus recursos financeiros. Sem ouro, sem exército.
- **Exército** — Sua força militar. Crucial para enfrentar Drakmar.
- **Popularidade** — O apoio do povo. Pode ser sua maior arma.
- **Sabedoria** — Conhecimento acumulado. Abre caminhos diplomáticos.

## Dicas

- Fale com todos os NPCs — cada um tem informações úteis.
- Examine os itens do inventário para descobrir seus usos.
- As escolhas têm consequências: não há opção "certa" para todos os finais.
- Salve com frequência usando o comando `salvar`.
- Se você ativar o áudio, use `repetir` caso perca alguma narração.
- Preste atenção nas conversas com o emissário e com o prisioneiro da masmorra.
- O prazo de Drakmar é real: o jogo termina em derrota após 25 turnos sem uma resolução.

## Finais

O jogo tem 5 finais diferentes dependendo das suas escolhas e dos seus atributos ao final. Explore caminhos alternativos para descobrir todos.
